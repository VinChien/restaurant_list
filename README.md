## Restaurant_list

這是練習使用 Node.js + Express 的網站。

## 使用者可以在首頁看到所有餐廳與它們的簡單資料：

  餐廳照片<br>
  餐廳名稱<br>
  餐廳分類<br>
  餐廳評分<br>
  
## 請先確認有安裝 node.js 與 npm<br>
將專案 clone 到本地<br>
在本地開啟之後，透過終端機進入資料夾，輸入： `npm install`<br>
安裝完畢後，繼續輸入： `npm run start`<br>
若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址: `Listening on http://localhost:3000`<br>
若欲暫停使用，輸入: `ctrl + c`
